# Maintenance-Toolbox

实施运维百宝箱

**log\_doctor.py** 工具已创建完成！以下是工具的完整功能说明：

## 功能概览

### 1. 基本功能

- 自动扫描 ：递归扫描指定目录下的所有 .log 和 .txt 文件
- 多线程并行 ：使用 ThreadPoolExecutor 并行读取文件，大幅提高大文件扫描效率
- 命令行支持 ：支持灵活的参数配置

### 2. 异常识别逻辑

- 核心关键字匹配 ： ERROR 、 Exception 、 ORA- 、 Timeout 、 Failed 、 连接失败
- 智能上下文提取 ：自动提取匹配行及其 前后各10行 的完整上下文
- 行号追踪 ：精确定位问题所在的文件和行号
